# Annoying Effects

一个Minecraft模组，加入了许多通常不致命但会使玩家感到难受的状态效果。这个模组将会用于[Lantern-in-Storm](https://github.com/BaicaiBear/Lantern-in-Storm/)中玩家在`unstable`区域的负面效果。

所有效果的效果都是不写明的，需要玩家自己探索发现。
